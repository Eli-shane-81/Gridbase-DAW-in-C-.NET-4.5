# Gridbase-DAW-in-C#.NET-4.5
A grid-based Digital Audio Workstation (DAW) built in C# (.NET Framework 4.5) using NAudio, focused on simple sequencing and audio playback with file export(MID,MP3,WAV) config with database.

This app is basicelly build for the users who want to learn DAW without worrying about all the complex functionalities and file management.
